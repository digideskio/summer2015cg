void Turntable::render ( )
virtual

Renders the turntable flat out on the zx-plane.

The turntable is created with 3 tapered cylinders using gluCylinder() function calls. The topmost cylinder has top radius set to zero giving an appearance of closed off lid to the base cylinder.

Reimplemented from Model.
